# Microsolvation of Protonated Glycine: Infrared Spectra from Data-Driven Quantum Many-Body Simulations

This repository contains the raw data of the paper mentioned above.

1. ```1-body-glyh+/MBX_files```: 1-body GlyH+ MB-nrg files necessary to implement the potential into MBX 
2. ```2-body-glyh+-h2o/MBX_files```: 2-body GlyH+-Water MB-nrg files necessary to implement the potential into MBX 
3. ```qc@mb-nrg-glyh+/graph_qc-compress.pb```: QC@MB-nrg_GlyH+ graph file
